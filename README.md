# Python Practice Fundamentals

This repository contains Python fundamentals practice and examples.

It covers core programming concepts through small, clear scripts.

## Topics Covered
- Variables and arithmetic operations
- Comparisons and boolean logic
- Conditional statements (if/else)
- Functions with parameters
- List operations and sorting
- Nested data structures
- Basic Object-Oriented Programming (OOP)

## Files
- `basics.py` — Variables, math operations, comparisons, and simple functions  
- `conditions.py` — Conditional logic and validation functions  
- `lists.py` — List operations, loops, sorting, and nested data  
- `oop.py` — Classes, objects, and methods  

## How to Run

Make sure Python is installed, then run:

```bash
python basics.py
python conditions.py
python lists.py
python oop.py
